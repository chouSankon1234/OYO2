I. HPを作成する方法

1. 自分でhtml文書を書く方法。自分でhtml文書を書き、cssやjsはネットのサンプルを利用する。html編集ソフトが利用できる。例、Visual Studio Code

2. ソフトを利用する方法。

例、AdobeのHPソフトDreamweaver（DW）、ホームページビルダー。

3. ホームページ制作サイトを利用する方法。

https://ja.wordpress.org/　、　https://ja.wix.com/のようなサイトで無料ホームページ制作ツールを利用してホームページ（HP）を作ることができる。無料公開（広告あり）できる。類似サイトが多い。メリット：HP作成ソフトはいらない。

4. HPテンプレートをダウンロードして利用する（著作権に注意）。HPソフトはいらない。今回はhtmlテンプレートをダウンロードして利用してみる。

5. AIを利用する。

6. Wordソフトなどを利用する。wordファイルをhtmlファイルに保存すれば、HP作成できる。（当ページはwordファイルで作成した。）

＜スマホで確認する＞

Google chromeブラウザを右クリックメニューで「このページのQRコード作成」でQRコード作成してスマホでページをみる。（学校のサーバーは読めないかも）

 

以下はQRコードの作成方法の説明。興味あれば自分でやってみてください。

1. 「QRコード作成」で検索して最上位の「QRコード[二次元バーコード]作成 【無料】」をクリックする。

https://www.cman.jp/QRcode/

2. ページのアドレスを「【1】QRコードを作成する文字を指定しよう 」の下の空欄にペストする→「QRコードを作成する」ボタンOK。

3. 表示されたQRコードをスマホでスキャンしてページを確認する。作成したQRコードをページに挿入して「スマホでご覧する方はスマホでQRコードを読み取ってください」と記入すれば、親切である。
